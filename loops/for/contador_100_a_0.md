# Contador_100_a_0

## Problemática
Se tiene que hacer un contador de 100 al 0

## Pseudocódigo
```
Inicio
    for (x = 100, x >= 0, x = x-1)
        print x
Fin
```

## Corridas de escritorio
Inicio  
    x = 100 // print 100  
    x = 99 // print 99  
    x = 98 // print 98  

    x = 1 // print 1  
    x = 0 // print 0  
Fin
